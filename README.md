# Student Performance Prediction System (FCNN)

A deep learning-based web application that predicts student performance using a Fully Connected Neural Network (FCNN). The model is deployed using Streamlit for real-time predictions.

---

##  Project Overview

This project predicts student performance based on various features such as:
- Study time
- Failures
- Family support
- School support
- Activities
- Internet access, etc.

The system uses a trained FCNN model and provides predictions through a simple web interface.

---

##  Model Details

- Model Type: Fully Connected Neural Network (FCNN)
- Framework: PyTorch
- Input Features: 58
- Layers:
  - Input Layer → 64 neurons
  - Hidden Layer → 32 neurons
  - Output Layer → 1 neuron
